# AI-Based Smart Traffic Monitoring & Congestion Prediction System

## How to Run

### Prerequisites
- Node.js (v16 or above) — download from https://nodejs.org

### Steps

1. Open terminal / command prompt in this folder
2. Install dependencies:
   ```
   npm install
   ```
3. Start the app:
   ```
   npm start
   ```
4. Opens automatically at http://localhost:3000

### Build for production
```
npm run build
```

## Project Info
- Model: Random Forest Classifier
- Dataset: Metro Interstate Traffic Volume (UCI)
- Accuracy: 94%
- Framework: React.js + Recharts
